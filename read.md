# AI Engineer Journey 🚀

This repository documents my 12-week journey to becoming an AI Engineer.

* Build strong Python and data engineering skills
* Understand ML fundamentals deeply
* Build real-world AI systems (LLMs, APIs, pipelines)

Each week focuses on a specific module:

* Week 1: Python Engineering
* Week 2: Pandas & Data Handling
* Week 3: PySpark
* Week 4: Machine Learning Basics
* Week 5: Backend + APIs
* Week 6: AI Systems

## Daily System

* Learn → Practice → Apply → Build → Project → Revise

## Outcome

* Strong portfolio projects

* Real AI engineering skills
