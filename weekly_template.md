📅 WEEK TEMPLATE (6 days)

Each day = 3 hours max (ADHD friendly)

🟢 Day 1 — Learn + Break
Learn core concept of module
Rewrite it in your own words

👉 Ask yourself:
“Where would this be used in a real system?”

🟢 Day 2 — Practice
Solve 5–10 small problems
Focus on mistakes

👉 Key rule:
Don’t just solve → understand patterns

🟢 Day 3 — Real Data / Application
Apply on dataset / real input

👉 Example:
Pandas → clean messy dataset

🟢 Day 4 — Build Component
Build 1 reusable piece

👉 Example:

data cleaner function
API wrapper
embedding generator
🟢 Day 5 — Mini Project (CRITICAL)
Combine everything

👉 This becomes portfolio

🟢 Day 6 — Notes + Interview Prep

You MUST create:

📘 1. Concept Notes
“What is groupby?”
“When to use dict vs list?”
🧠 2. Mistake Notes
bugs you made
why they happened
💼 3. Project Story
Problem
Solution
Trade-offs
